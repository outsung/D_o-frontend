## 기능

- `React` + `Typescript` + `node js`
- 이메일 인증 + jwt토큰 기반 자체 로그인 기능
- `mocha`를 사용한 서버 단위 테스트
- `three js`, `r3f`, `cannon`, `spring`을 사용한 3D 인터렉티브 기능
- `matter js`, `google mediapipe`를 사용한 2D 인터렉티브 기능
- `github action`을 사용한 버전관리, DC 기능
